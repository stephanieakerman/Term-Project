How does the bot work?

To get started with the bot, follow these steps:

  1. Fork this repository HERE.
  2. Clone the forked repository to your local machine using GitHub Desktop.
  3. Ensure you have Python installed on your system along with the necessary dependencies. You can do this by running the following commands in your terminal:
pip install python-telegram-bot
pip install openai
pip install flask

Once the dependencies are installed, navigate to the cloned directory and execute the RUN.PY script. This script will initiate both the bot and the associated website functionalities, allowing seamless interaction.

Once the user has run the code, follow these simple steps to open the bot in Telegram:

  1. Click HERE to open a chat with the bot in Telegram.
  2. Once the chat window is open, start the conversation by pressing 'start'.
  3. The bot will reply with a prompt, guiding you on how to proceed further.
  4. Depending on the features you wish to use, input the appropriate commands as instructed by the bot: Send a text to convert it to speech, or an audio message to convert it to text!
  5. Enjoy the experience and let our bot help you with your communication tasks!
